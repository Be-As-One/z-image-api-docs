## Create Your First Image

<Steps>
  <Step title="Set up your environment">
    Store your API key securely. For testing, you can use environment variables:

    ```bash
    export Z_IMAGE_API_KEY="your_api_key_here"
    ```
  </Step>

  <Step title="Create a task">
    Send a POST request to create an image generation task:

    <CodeGroup>
    ```bash cURL
    curl -X POST "https://z-image.vip/api/v1/jobs/createTask" \
      -H "Authorization: Bearer $Z_IMAGE_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "seedream-text-to-image",
        "inputs": {
          "prompt": "A serene Japanese garden with cherry blossoms",
          "aspectRatio": "16:9",
          "resolution": "2k"
        }
      }'
    ```

    ```javascript Node.js
    const response = await fetch('https://z-image.vip/api/v1/jobs/createTask', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${process.env.Z_IMAGE_API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        model: 'seedream-text-to-image',
        inputs: {
          prompt: 'A serene Japanese garden with cherry blossoms',
          aspectRatio: '16:9',
          resolution: '2k'
        }
      })
    });

    const data = await response.json();
    console.log('Task ID:', data.data.taskId);
    ```

    ```python Python
    import requests
    import os

    response = requests.post(
        'https://z-image.vip/api/v1/jobs/createTask',
        headers={
            'Authorization': f'Bearer {os.getenv("Z_IMAGE_API_KEY")}',
            'Content-Type': 'application/json'
        },
        json={
            'model': 'seedream-text-to-image',
            'inputs': {
                'prompt': 'A serene Japanese garden with cherry blossoms',
                'aspectRatio': '16:9',
                'resolution': '2k'
            }
        }
    )

    data = response.json()
    print('Task ID:', data['data']['taskId'])
    ```
    </CodeGroup>

    You'll receive a response like:
    ```json
    {
      "code": 200,
      "message": "success",
      "data": {
        "taskId": "task_clxxxxxx",
        "orderId": "ord_clxxxxxx",
        "creditsUsed": 5
      }
    }
    ```

    <Warning>
    ‚ö†Ô∏è **Critical:** A 200 response only means the task was created, not completed. You must poll the status endpoint to get the final result.
    </Warning>
  </Step>

  <Step title="Check task status">
    Poll the status endpoint until the task completes:

    <CodeGroup>
    ```bash cURL
    curl -X GET "https://z-image.vip/api/v1/jobs/recordInfo?taskId=task_clxxxxxx" \
      -H "Authorization: Bearer $Z_IMAGE_API_KEY"
    ```

    ```javascript Node.js
    async function waitForCompletion(taskId) {
      while (true) {
        const response = await fetch(
          `https://z-image.vip/api/v1/jobs/recordInfo?taskId=${taskId}`,
          {
            headers: {
              'Authorization': `Bearer ${process.env.Z_IMAGE_API_KEY}`
            }
          }
        );

        const data = await response.json();
        const state = data.data.state;

        if (state === 'success') {
          const result = JSON.parse(data.data.resultJson);
          return result.resultUrls;
        } else if (state === 'fail') {
          throw new Error('Task failed');
        }

        await new Promise(resolve => setTimeout(resolve, 3000));
      }
    }
    ```

    ```python Python
    import time
    import json

    def wait_for_completion(task_id):
        while True:
            response = requests.get(
                f'https://z-image.vip/api/v1/jobs/recordInfo?taskId={task_id}',
                headers={
                    'Authorization': f'Bearer {os.getenv("Z_IMAGE_API_KEY")}'
                }
            )

            data = response.json()
            state = data['data']['state']

            if state == 'success':
                result = json.loads(data['data']['resultJson'])
                return result['resultUrls']
            elif state == 'fail':
                raise Exception('Task failed')

            time.sleep(3)
    ```
    </CodeGroup>
  </Step>

  <Step title="Get your result">
    Once the task completes, you'll receive the image URL:

    ```json
    {
      "code": 200,
      "message": "success",
      "data": {
        "taskId": "task_clxxxxxx",
        "state": "success",
        "resultJson": "{\"resultUrls\":[\"https://...\"]}"
      }
    }
    ```
  </Step>
</Steps>

## FAQ

**How long does generation take?**
- Images: 10-30 seconds
- Videos: 30 seconds - 2 minutes

**What if I get 'Insufficient credits'?**
Add credits in your [dashboard](https://z-image.vip/admin). Image generation costs 5 credits, videos cost 15-22 credits.

**Can I regenerate the same image?**
Yes! Use the same `seed` value in your request to get consistent results.
